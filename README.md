## 🚀 Quick Start

### Prerequisites
> 📋 Ensure you have the following installed:
> - Node.js (v14+)
> - MongoDB
> - npm or yarn

### Installation Steps

```bash
# 1. Clone the repository
git clone https://github.com/Hirumitha-Kuladewa/book-review-system.git

# 2. Setup Backend
cd backend
npm install
npm run dev

# 3. Setup Frontend
cd frontend
npm install
npm run dev
```
